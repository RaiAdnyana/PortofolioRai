Tentang

Saya

Perkenalkan Saya I Made Rai Adnyana, Saya berasal dari Banjar Batunya, Kecamatan Baturiti, Kabupaten Tabanan. Saya lahir di Batunya pada tanggal 5 Maret 2007. Saya tumbuh dalam keluarga yang penuh kasih sayang, di mana orang tua saya selalu mendukung dan menginspirasi saya untuk mengejar impian saya. Sejak kecil, saya selalu memiliki keinginan untuk menjelajahi dunia dan belajar hal-hal baru.

Pendidikan saya dimulai di SD Negeri 1 Batunya, dimana saya memulai untuk belajar dan mengembangkan diri saya, setelah 6 Tahun sekolah di SD Negeri 1 Batunya saya kemudian melanjutkan Pendidikan saya ke jenjang SMP. Saya melanjutkan Pendidikan saya di SMP Negeri 1 Baturiti, di SMP saya bertemu dengan banyak teman baru. 3 Tahun telah belalu, sekarang Saya telah bersekolah di jenjang Menengah. Sekarang Saya bersekolah di SMK NEGERI 1 DENPASAR, dengan jurusan Rekayasa Perangkat Lunak.

Saya sangat tertarik dengan dunia Pemrograman. Saya belajar Pemrograman mulai saat sekolah menengah, saat ini Saya sudah menguasai beberapa bahasa Pemrograman seperti html dan css. Saya masih aktif mempelajari Pemrograman dan sedang mempelajari bahasa JavaScript dan Framework Django.
